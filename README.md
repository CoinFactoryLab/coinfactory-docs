# CoinFactory Docs

CoinFactory is a multi-chain platform for launching memecoins, creating tokens, and managing liquidity across multiple blockchain ecosystems.

The platform provides tools for creating tokens, launching memecoins, distributing tokens, and managing token supply across EVM networks, Solana, TON, and Sui.

CoinFactory allows founders, developers, and communities to launch tokens and manage token operations without writing smart contracts.

---

# What is CoinFactory

CoinFactory is a token launch and token management platform that supports multiple blockchain networks.

With CoinFactory users can:

- launch memecoins
- create fungible tokens
- distribute tokens using multisender tools
- create and manage liquidity pools
- mint or burn tokens
- revoke permissions and secure token ownership
- manage token metadata and token pages

The platform is designed for memecoin launches, tokenization projects, and community token ecosystems.

# Solana Services

CoinFactory provides multiple tools for managing SPL tokens on the Solana ecosystem.

Supported features include:

### SPL Token Creation

Users can create SPL tokens for:

- memecoins
- tokenized assets
- NFT ecosystems
- community tokens

### Token Permission Management

CoinFactory allows revoking and managing token permissions:

- revoke mint authority
- revoke freeze authority
- revoke metadata update authority

These tools help secure Solana tokens after launch.

### Token Distribution

Solana token distribution tools include:

- SPL token multisender
- batch token transfers

These tools are useful for airdrops and token distribution.

### Liquidity Management

CoinFactory provides tools for working with liquidity on Solana DEX platforms.

Supported features:

- create Raydium liquidity pools
- add liquidity
- remove liquidity

### Token Supply Management

Users can manage token supply with:

- mint tokens
- burn tokens

### Token Account Management

Advanced token account tools:

- freeze token accounts
- unfreeze token accounts

### Token Pages

CoinFactory allows creating custom token pages that include token information and launch details.

# EVM Network Services

CoinFactory provides token launch tools across multiple EVM networks.

These tools support memecoin launches, tokenized assets, and community tokens.

## Token Creation

Users can create ERC-20 tokens using multiple templates:

- Standard
- Essential
- Taxable

These templates allow launching tokens with different tokenomics models.

Supported use cases:

- memecoins
- utility tokens
- governance tokens
- tokenized assets

## Token Distribution

CoinFactory provides token distribution tools:

- ERC-20 multisender
- batch token transfers

These tools are commonly used for airdrops and community distributions.

## Liquidity Pools

CoinFactory supports liquidity pool creation on Uniswap-compatible exchanges.

Supported operations:

- create Uniswap V2 liquidity pool
- create Uniswap V3 liquidity pool
- add liquidity to existing pool
- remove liquidity

## Token Ownership Management

Security tools include:

- revoke ownership
- transfer ownership

These tools help secure tokens after deployment.

## Token Supply Management

Supported operations:

- mint tokens
- burn tokens

## Token Control

Administrative controls include:

- pause token transfers
- unpause token transfers

## Token Pages

CoinFactory provides tools for generating custom token pages for token projects.

These pages can be used to present token information and project details.
