# What Is an SPL Token

An SPL token is a fungible token created using the Solana Program Library (SPL).

SPL tokens are similar to ERC20 tokens on Ethereum, but are designed for the Solana blockchain.

They are commonly used for:

- memecoins
- utility tokens
- community tokens
- governance tokens
- tokenized assets

Many Solana projects launch SPL tokens as the core asset of their ecosystem.

# Steps to Create an SPL Token

Creating an SPL token usually involves several steps.

## 1. Define Token Parameters

When creating an SPL token you typically configure:

- token name
- token symbol
- total supply
- token metadata
- token decimals

These parameters define how the token behaves on the Solana blockchain.

## 2. Deploy the Token

After defining token parameters, the next step is deploying the SPL token on the Solana network.

Deploying a token creates the token mint account and registers the token on the blockchain.

Token deployment can be done using developer tools or token launch platforms.

CoinFactory allows users to deploy SPL tokens without writing code.

---

## 3. Configure Token Authorities

SPL tokens include several authorities that control token permissions.

Common authorities include:

- mint authority
- freeze authority
- metadata update authority

These permissions allow token creators to manage supply and token behavior.

Many projects revoke these authorities after launch to increase transparency.

---

## 4. Distribute Tokens

Once the token is created, it can be distributed to users.

Common distribution methods include:

- airdrops
- community rewards
- token sales
- contributor allocations

Multisender tools can help distribute tokens to many wallets at once.

---

## 5. Launch Liquidity

Many tokens are listed on decentralized exchanges after launch.

Popular Solana DEX platforms include:

- Raydium
- Orca

Liquidity pools allow users to trade the token in open markets.
