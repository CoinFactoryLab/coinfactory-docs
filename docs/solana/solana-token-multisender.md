# What Is a Solana Token Multisender

A Solana token multisender allows token creators to distribute SPL tokens to many wallets without sending transactions one by one.

Instead of manually transferring tokens to each address, the multisender processes token transfers in batches.

This significantly simplifies token distribution for Solana projects.

Solana multisender tools are commonly used for:

- token airdrops
- community rewards
- presale distributions
- contributor payments
- marketing campaigns

---

# What Are SPL Tokens

SPL tokens are the standard token format used on the Solana blockchain.

They function similarly to ERC20 tokens on Ethereum.

SPL tokens are widely used for:

- memecoins
- utility tokens
- governance tokens
- community tokens

Many Solana memecoins are launched as SPL tokens.

# How to Send SPL Tokens to Multiple Wallets

Sending tokens to multiple wallets usually involves several steps.

## 1. Prepare the Wallet List

The first step is preparing a list of wallet addresses that will receive tokens.

This list may include:

- airdrop participants
- community members
- presale buyers
- contributors

Wallet addresses are typically uploaded as a list or CSV file.

---

## 2. Select the SPL Token

Next, select the SPL token that will be distributed.

The token must already exist on the Solana network.

---

## 3. Configure Token Distribution

Token creators configure how tokens will be distributed.

Distribution options include:

- equal token amounts
- custom allocations per wallet

---

## 4. Execute the Batch Transfer

After configuring the distribution, the multisender executes the batch transfer.

This allows sending tokens to many wallets quickly and efficiently.
