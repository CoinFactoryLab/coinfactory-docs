A Sui multisender is a tool that allows users to send SUI coins or Sui tokens to multiple wallet addresses in a single batch process.

Multisender tools are commonly used for token airdrops, community rewards, and large-scale token distributions.

# What Is a Sui Multisender

A Sui multisender allows users to distribute coins or tokens to many wallets without sending transactions individually.

Instead of manually sending assets to each address, the multisender processes transfers in batches.

This simplifies large token distributions for blockchain projects and communities.

Sui multisender tools are commonly used for:

- token airdrops
- community rewards
- contributor payments
- presale distributions
- marketing campaigns

# How to Send SUI to Multiple Wallets

Sending SUI or tokens to multiple wallets usually involves several steps.

## 1. Prepare the Wallet List

First prepare the list of recipient wallet addresses.

This list may include:

- community members
- airdrop participants
- presale buyers
- contributors

Wallet lists are typically uploaded as a CSV file or pasted as a list of addresses.

---

## 2. Select the Asset

Next select the asset that will be distributed.

This can be:

- SUI coins
- tokens created on the Sui network

---

## 3. Configure Distribution

Define how the assets will be distributed.

Distribution options include:

- equal amounts for each wallet
- custom amounts per wallet

---

## 4. Execute the Batch Transfer

After configuring the distribution, the multisender executes the batch transfer.

This allows sending assets to many wallets efficiently.
