# What Is an ERC20 Token

An ERC20 token is a fungible cryptocurrency token created using the ERC20 standard on Ethereum.

The ERC20 standard defines how tokens behave and interact with wallets, smart contracts, and decentralized exchanges.

ERC20 tokens are widely used across the EVM ecosystem for:

- memecoins
- utility tokens
- governance tokens
- DeFi tokens
- tokenized assets

Many popular cryptocurrencies are ERC20 tokens.

# Steps to Create an ERC20 Token

Creating an ERC20 token usually involves several steps.

## 1. Choose a Network

The first step is selecting the blockchain network where the token will be deployed.

Different networks offer different transaction fees and ecosystems.

Popular options include Ethereum, Base, BNB Chain, and Arbitrum.

---

## 2. Configure Token Parameters

When creating an ERC20 token you typically define:

- token name
- token symbol
- total supply
- token decimals

These parameters define how the token will behave once deployed.

You can configure token parameters using the CoinFactory token generator:

https://coinfactory.app/generator

---

## 3. Deploy the Token

After configuring parameters, the token contract is deployed to the blockchain.

Deployment creates the token smart contract and registers the token on the selected network.

CoinFactory allows users to deploy ERC20 tokens without writing smart contracts.

---

## 4. Launch Liquidity

After the token is deployed, liquidity pools are usually created on decentralized exchanges.

Popular DEX platforms include:

- Uniswap
- PancakeSwap
- SushiSwap

Liquidity pools allow users to trade the token.

---

## 5. Manage the Token

After deployment, token creators often manage token permissions and supply.

Common actions include:

- minting tokens
- burning tokens
- revoking ownership
- pausing token transfers

CoinFactory provides tools for managing ERC20 tokens after launch.
