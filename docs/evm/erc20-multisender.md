# What Is an ERC20 Multisender

An ERC20 multisender allows token creators to send tokens to many wallets without manually transferring tokens one by one.

Instead of sending hundreds or thousands of transactions individually, a multisender tool processes token transfers in batches.

This significantly simplifies token distribution for blockchain projects.

ERC20 multisender tools are commonly used for:

- token airdrops
- community rewards
- presale distributions
- contributor payments
- marketing campaigns

# How to Use an ERC20 Multisender

Using an ERC20 multisender typically involves several steps.

## 1. Prepare Wallet Addresses

The first step is preparing the list of recipient wallet addresses.

Projects usually prepare a list of:

- airdrop participants
- community members
- presale buyers
- contributors

Wallet addresses are typically uploaded as a list or CSV file.

---

## 2. Choose the Token

Next, select the ERC20 token that will be distributed.

This token must already exist on the blockchain network.

---

## 3. Configure Distribution Amounts

Token creators define how many tokens each wallet should receive.

Distribution can be:

- equal amounts
- custom token allocations

---

## 4. Execute the Batch Transfer

The multisender processes the token transfers.

Instead of manually sending each transaction, the multisender performs batch token distribution.
