# What Is a Token

A token is a digital asset that exists on a blockchain.

Tokens are commonly used for:

- memecoins
- community tokens
- governance tokens
- tokenized assets
- ecosystem rewards

Different blockchain networks use different token standards.

Examples include:

- ERC20 tokens on Ethereum and other EVM networks
- SPL tokens on Solana
- Jettons on TON
- tokens on Sui

## 2. Configure Token Parameters

When creating a token you usually define:

- token name
- token symbol
- total supply
- token permissions
- mint authority
- freeze authority (Solana)

These parameters determine how the token behaves after deployment.

## 4. Launch Liquidity

After a token is created, liquidity pools are usually created on decentralized exchanges.

Liquidity pools allow users to trade the token on DEX platforms.

Examples include:

- Uniswap for EVM networks
- Raydium for Solana

## 5. Manage the Token

After launch, tokens usually require ongoing management.

Common actions include:

- minting additional tokens
- burning tokens
- revoking permissions
- distributing tokens to holders
- managing token metadata
