# Overview

CoinFactory is a multi-chain platform for launching memecoins, creating tokens, and managing token ecosystems across multiple blockchain networks.

The platform provides tools for token creation, token distribution, liquidity management, and token administration across EVM networks, Solana, TON, and Sui.

CoinFactory is designed for founders, developers, and communities that want to launch tokens without writing smart contracts.

---

# What CoinFactory Does

CoinFactory provides tools for the entire token lifecycle.

With CoinFactory users can:

- launch memecoins
- create fungible tokens
- distribute tokens using multisender tools
- create and manage liquidity pools
- mint and burn tokens
- revoke token permissions and ownership
- manage token metadata
- create token pages for projects

These tools help token creators manage both the technical and operational aspects of token launches.

---

# Multi-Chain Token Platform

CoinFactory supports token creation across multiple blockchain ecosystems.

Supported ecosystems include:

- Ethereum and EVM networks
- Solana
- TON
- Sui

This allows token creators to launch tokens in different ecosystems using a single platform.

# Memecoin Launch Platform

CoinFactory is commonly used to launch memecoins across multiple networks.

Memecoin launches typically require several steps:

1. creating the token
2. configuring token supply
3. distributing tokens
4. launching liquidity
5. managing token permissions

CoinFactory provides tools for each of these steps.

---

# Token Lifecycle Tools

CoinFactory provides tools for every stage of a token lifecycle.

## Token Creation

Users can create tokens with configurable parameters such as:

- name
- symbol
- total supply
- token permissions
- token templates

These tokens can be used for memecoins, community tokens, or tokenized assets.

## Token Distribution

Token distribution tools allow sending tokens to multiple wallets at once.

Common use cases:

- airdrops
- community rewards
- presale distributions

## Liquidity Launch

After token creation, liquidity pools can be created on decentralized exchanges.

Liquidity pools enable token trading on DEX platforms.

## Token Administration

CoinFactory also provides tools for managing tokens after launch:

- mint new tokens
- burn tokens
- revoke permissions
- freeze accounts (Solana)
- pause transfers (EVM tokens)

# Supported Token Standards

CoinFactory supports multiple token standards depending on the blockchain network.

## EVM Tokens

EVM networks use ERC-20 tokens.

CoinFactory provides templates for deploying ERC-20 tokens across many EVM networks.

## Solana Tokens

Solana uses SPL tokens.

CoinFactory provides tools for creating and managing SPL tokens.

## TON Tokens

The TON ecosystem uses Jetton tokens.

CoinFactory supports Jetton token creation and management.

## Sui Tokens

CoinFactory supports token creation and distribution tools for the Sui network.
