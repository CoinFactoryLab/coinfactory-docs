# What Is a Jetton

A Jetton is a fungible token created on the TON blockchain.

Jettons function similarly to ERC20 tokens on Ethereum or SPL tokens on Solana.

They can represent different types of digital assets such as:

- memecoins
- community tokens
- utility tokens
- governance tokens
- tokenized assets

Many projects launching in the TON ecosystem use Jettons as their main token standard.

# Steps to Create a Jetton

Creating a Jetton on TON usually involves several steps.

## 1. Configure Token Parameters

The first step is defining the token parameters.

Typical parameters include:

- token name
- token symbol
- total supply
- token metadata

These parameters define how the token behaves once deployed on the TON network.

## 2. Deploy the Jetton

After configuring the token parameters, the Jetton smart contract is deployed on the TON blockchain.

Deployment registers the token and makes it available for transfers and integrations.

CoinFactory allows users to deploy Jettons without writing smart contracts.

---

## 3. Distribute Tokens

After the token is deployed, it can be distributed to users.

Common distribution methods include:

- airdrops
- community rewards
- contributor allocations
- presale distributions

Batch distribution tools can simplify sending tokens to many wallets.

---

## 4. Manage Token Supply

After launch, token creators may manage token supply and metadata.

Common token operations include:

- minting additional tokens
- burning tokens
- updating token metadata

These tools allow projects to manage their token ecosystem after launch.
