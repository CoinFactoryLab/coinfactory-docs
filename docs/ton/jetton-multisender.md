# What Is a Jetton Multisender

A Jetton multisender allows token creators to distribute Jetton tokens to many wallets without sending transfers individually.

Instead of manually sending tokens to each address, the multisender processes token transfers in batches.

This simplifies token distribution for projects built on the TON blockchain.

Jetton multisender tools are commonly used for:

- token airdrops
- community rewards
- contributor payments
- presale distributions
- marketing campaigns

---

# What Are Jettons

Jettons are fungible tokens on The Open Network (TON).

They function similarly to ERC20 tokens on Ethereum and SPL tokens on Solana.

Jettons are commonly used for:

- memecoins
- community tokens
- governance tokens
- utility tokens
- tokenized assets

Many projects in the TON ecosystem launch Jettons as their primary token.

# How to Send Jettons to Multiple Wallets

Sending Jetton tokens to multiple wallets usually involves several steps.

## 1. Prepare the Wallet List

The first step is preparing the list of recipient wallet addresses.

This list may include:

- community members
- airdrop participants
- presale buyers
- contributors

Wallet addresses are typically uploaded as a CSV file or pasted as a list.

---

## 2. Select the Jetton

Next select the Jetton token that will be distributed.

The token must already exist on the TON network.

---

## 3. Configure Token Distribution

Token creators configure how tokens will be distributed.

Distribution options include:

- equal token amounts
- custom token allocations for each wallet

---

## 4. Execute the Batch Transfer

After configuring the distribution, the multisender executes the batch transfer.

This allows sending Jetton tokens to many wallets efficiently.
